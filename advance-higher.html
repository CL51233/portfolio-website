SQA Advance Higher Graphic Communication Project

Project Status : In Progress

The project consists of two distinct sections: CVMG (Creative Visual Media Graphics) and Technical Graphics (TG).

CVMG focuses on digital graphic design elements, including user interfaces, posters, and other forms of visual communication. This section of the project will feature advertisements for the game, as well as the game's user interface, all aligned with a cohesive visual identity inspired by low-poly style games.

TG (Technical Graphics) emphasizes the creation of 3D models and technical sketches. The centerpiece of this section is a 3D scene representing the game's main menu, designed with low-poly aesthetics. This scene serves as the technical backbone of the game's visual style.

Tools: Inventor, MS Publisher, Photoshop
